=====================
Starting from scratch
=====================

The `csdfpy` module is not just designed for deserializing and serializing
`.csdf` or `.csdfe` files. It can also be used to create new datasets.
This is particularly useful when serializing a raw or a processed
data as a `.csdf` or `.csdfe` file.
