==================================
Welcome to csdfpy's documentation!
==================================

The ``csdfpy`` is a python module for handling a variety of scientific datasets
stored using the core scientific dataset (CSD) model. The two supported file
extensions are ``.csdf`` and ``.csdfx``. This module is build on the idea of
a portable scientific dataset format which contains the (CSD) model.
