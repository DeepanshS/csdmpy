An arbitrarily sampled grid controlled dimension.

Warning

This class should not be used directly. Instead, use the CSDModel object to access the attributes and methods of this class. See example, Arbitrarily sampled grid dimension controlled variable.

This class returns an object which represents a physical controlled variable, sampled arbitrarily along a grid dimension. Let \(\mathbf{A}_k\) be an ordered array of physical quantities, \(c_k\) be the reference offset, and \(o_k\) be the origin offset along the \(k^{th}\) grid dimension, then the coordinates along this dimension are given as

\[\mathbf{X}_k^\mathrm{ref} = \mathbf{A}_k - c_k \mathbf{1},\]
\[\mathbf{X}_k^\mathrm{abs} = \mathbf{X}_k^\mathrm{ref} + o_k \mathbf{1},\]

where \(\mathbf{X}_k^\mathrm{ref}\) is an ordered array of the reference controlled variable coordinates, \(\mathbf{X}_k^\mathrm{abs}\) is an ordered array of the absolute controlled variable coordinates, and \(\mathbf{1}\) is an array of ones.
