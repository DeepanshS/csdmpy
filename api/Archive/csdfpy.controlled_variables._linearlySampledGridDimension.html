A linearly sampled grid controlled dimension.

Warning

This class should not be used directly. Instead, use the CSDModel object to access the attributes and methods of this class. See example, Linearly sampled grid controlled variable.

This class returns an object which represents a physical controlled variable, sampled linearly along a grid dimension. Let \(m_k\) be the sampling interval, \(N_k \ge 1\) be the number of points, \(c_k\) be the reference offset, and \(o_k\) be the origin offset along the \(k^{th}\) grid dimension, then the coordinates along the grid dimension are given as

\[\mathbf{X}_k^\mathrm{ref} = [m_k j ]_{j=0}^{N_k-1} - c_k \mathbf{1},\]
\[\mathbf{X}_k^\mathrm{abs} = \mathbf{X}_k^\mathrm{ref} + o_k \mathbf{1},\]

where \(\mathbf{X}_k^\mathrm{ref}\) is an ordered array of the reference controlled variable coordinates, \(\mathbf{X}_k^\mathrm{abs}\) is an ordered array of the absolute controlled variable coordinates, and \(\mathbf{1}\) is an array of ones.
