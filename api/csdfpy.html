csdfpyÂ¶

The csdfpy is a python module for importing and exporting the CSD model file exchange formats. The module is based on the core scientific dataset (CSD) model which is designed as a building block in the development of a more sophisticated portable scientific dataset file standard. The CSD model supports \(p\)-component independent variable, \(y\), which is discretely sampled at \(N\) points in a \(d\)-dimensional independent variable space \((x_0, x_1, ... x_k, ... x_{d-1})\). The model also supports multiple dependent variables that are simultaneously sampled over the \(d\)-dimensional independent variable space.

In the csdfpy module, every instance of the CSDModel class holds a list of independent and dependent variables. Every independent variable, \(x_k\), is an instance of the IndependentVariable class while every dependent variable, \(y\), is an instance of the DependentVariable class. csdfpy.load(filename=None)Â¶

Opens a .csdf or .csdfe file and returns an instance of the CSDModel class.

The file must be a JSON serialization of the CSD Model.

>>> import csdfpy as cp
>>> data1 = cp.load('local_address/file.csdf')  
>>> data2 = cp.load('url_address/file.csdf')
Params:filename: A local or remote address to the .csdf or `.csdfe file.
Returns:A CSDModel instance.
csdfpy.new()Â¶

Returns an instance of the CSDModel class corresponding a \(0\mathrm{D}\{0\}\) dataset.

>>> import csdfpy as cp
>>> emptydata = cp.new()
>>> print(emptydata.data_structure)
{
  "CSDM": {
    "version": "0.0.9",
    "independent_variables": [],
    "dependent_variables": []
  }
}
Returns:A CSDModel instance
