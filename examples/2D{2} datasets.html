2D{2} datasetsÂ¶

The 2D{2} datasets have two independent variables, \(d=2\), and one two-component dependent variable, \(p=2\).

Vector datasetÂ¶

The following is an example of a simulated electric field dataset of a dipole as a function of two spatial dimensions.

>>> import csdfpy as cp

>>> filename = '../../test-datasets0.0.9/vector/electricField/electric_field_raw.csdfe'
>>> vectordata = cp.load(filename)
>>> print (vectordata.data_structure)
{
  "CSDM": {
    "version": "0.0.9",
    "independent_variables": [
      {
        "type": "linearly_sampled",
        "number_of_points": 64,

        "reference_offset": "2.0 cm",

        "label": "x",
        "reciprocal": {
          "quantity": "wavenumber"
        }

        "reference_offset": "2.0 cm",

The spatial dimensions are sampled linearly along the two independent variables. The tuples of independent and dependent variable instances from this example are

>>> x = vectordata.independent_variables
>>> y = vectordata.dependent_variables

with the respective coordinates (viewed only up to five values).

>>> print(x[0].coordinates[:5])
[-2.     -1.9375 -1.875  -1.8125 -1.75  ] cm

>>> print(x[1].coordinates[:5])
[-2.     -1.9375 -1.875  -1.8125 -1.75  ] cm

respectively. The components of the dependent variable are of a vector dataset. This is also seen from the quantity_type attribute of the corresponding dependent variable instance.

>>> print(y[0].quantity_type)
vector_2

Letâ€™s plot the vector data. To do this, we use the streamplot method from the matplotlib package. Before we could visualize, however, there is an initial processing step. We use the methods from the Numpy library for processing.

>>> import numpy as np

>>> X, Y = np.meshgrid(x[0].coordinates, x[1].coordinates)
>>> U, V = y[0].components[0], y[0].components[1]
>>> R = np.sqrt(U**2 + V**2)
>>> R/=R.min()
>>> Rlog=np.log10(R)

And now, the plot.

>>> import matplotlib.pyplot as plt

>>> fig, ax = plt.subplots(1,1, figsize=(5.4,5))
>>> ax.streamplot(X.value, Y.value, U, V, density =1,
...               linewidth=Rlog, color=Rlog, cmap='viridis')  

>>> ax.set_xlim([x[0].coordinates[0].value,
...             x[0].coordinates[-1].value])  
>>> ax.set_ylim([x[1].coordinates[0].value,
...             x[1].coordinates[-1].value])  

>>> # Set axes labels and figure title.
>>> ax.set_xlabel(x[0].axis_label)  
>>> ax.set_ylabel(x[1].axis_label)  
>>> ax.set_title(y[0].name) 

>>> # Set grid lines.
>>> ax.grid(color='gray', linestyle='--', linewidth=0.5)

>>> plt.tight_layout(pad=0., w_pad=0., h_pad=0.)
>>> plt.subplots_adjust(wspace=0.025, hspace=0.05)
>>> plt.savefig(vectordata.filename+'.pdf')
>>> plt.show()
